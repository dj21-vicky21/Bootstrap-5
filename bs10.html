Use the class="btn-lg" for large buttons or class="btn-sm" for small buttons:

Active/Disable button

the button is active, class="btn btn-primary active"


the button is disabled, class="btn btn-primary disabled"

Spinner border Buttons:

1. You can also add "spinners" to a button class="spinner-border "
2. its default size use small size for class = spinner-border-sm
3. Also use disable property value

spinner grow Buttons:

1. You can also add "glow" to a button class="glow-border "
2. its default size use small size for class = glow-border-sm
3. Also use disable property value
